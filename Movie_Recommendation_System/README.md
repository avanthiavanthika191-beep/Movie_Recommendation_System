# 🎬 Movie Recommendation System

## Project Description
The Movie Recommendation System is a Python-based application that recommends movies based on a user's selected movie. It uses the MovieLens dataset and Content-Based Filtering with Cosine Similarity to find similar movies.

## Features
- Import MovieLens dataset
- Clean and preprocess data
- Calculate movie similarity using Cosine Similarity
- Recommend the Top 5 similar movies
- Display recommended movies

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- VS Code

## Dataset
MovieLens Latest Small Dataset

Files:
- movies.csv
- ratings.csv
- links.csv
- tags.csv
